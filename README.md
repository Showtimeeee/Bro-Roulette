# Bro Roulette

Веб-приложение «Рулетка» — рандомайзер вариантов с анимированным барабаном.

## Как пользоваться

1. Добавь свои варианты в панель под барабаном
2. Нажми **КРУТИТЬ** (или кликни по колесу)
3. Барабан раскрутится и стрелка укажет на результат
4. Результат появится в блоке **История** с временной меткой

## Возможности

- Добавление и удаление вариантов (до 20 штук)
- Кнопка **Удалить все** — очистка вариантов одним нажатием
- Кнопка **Сбросить** — возврат к дефолтным вариантам
- Плавная анимация вращения с замедлением
- Стрелка-индикатор результата
- Цветные сегменты с текстом
- **История бросков** — последние 20 результатов(localStorage)
- Адаптивный дизайн (мобильные устройства)
